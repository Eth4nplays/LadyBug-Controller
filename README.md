# 🐞 LadyBug Remote
An open-source Bluetooth remote controller for the RoboForce LadyBug robot, built using MIT App Inventor 2.

For more information about LadyBug, click [here](https://roboforce.com.my/helloladybug/).

## ✨ Features
- 🎮 Game-style controller for easy robot movement control
- 📱 Simple and modern user interface
- 📡 Serial Bluetooth communication (send & receive data)
- 📋 Built-in instructions for easy setup and use

## 💾 Installation

- Go to **Releases**
- Download the latest `Ladybug.apk`

Or download directly: https://github.com/Eth4nplays/LadyBug-Controller/releases/latest/download/Ladybug.apk

## 💻 Source Code

This project is built using **MIT App Inventor 2**.

To open the source:
- Download `Ladybug.aia` from this repository
- Go to: https://ai2.appinventor.mit.edu/
- Import the `.aia` file to edit or rebuild the app
